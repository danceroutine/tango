# Getting Started

Welcome to Tango! This guide will help you get started with the framework.

## What is Tango?

Tango is a TypeScript web framework inspired by Django and Django REST Framework (DRF). It brings the elegance and productivity of Django's design patterns to the TypeScript ecosystem.

## Prerequisites

Before you begin, make sure you have the following installed:

- Node.js 22 or higher
- pnpm 9 or higher
