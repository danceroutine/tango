# Installation

## Using pnpm (Recommended)

```bash
pnpm add @danceroutine/tango-core
```

## Using npm

```bash
npm install @danceroutine/tango-core
```

## Using yarn

```bash
yarn add @danceroutine/tango-core
```

## Requirements

- Node.js >= 22.0.0
- TypeScript >= 5.6.0
